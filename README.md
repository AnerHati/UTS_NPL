 UTS NLP - Analisis Sentimen Kebijakan PPKM
Proyek ini adalah sistem klasifikasi sentimen teks Twitter berbahasa Indonesia terkait kebijakan PPKM menggunakan metode representasi Bag-of-Words (BoW), TF-IDF, dan algoritme Naive Bayes.

Cara Menjalankan Proyek:
1. Unduh file notebook `.ipynb` dan file dataset `.csv` dari repositori ini.
2. Buka [Google Colab](https://colab.research.google.com/) dan unggah file `.ipynb` tersebut.
3. Jalankan kode secara berurutan (Run).
4. Pada bagian "Tahap 2" yang memuat dataset, akan muncul tombol "Choose Files". Klik tombol tersebut dan unggah file dataset `.csv` yang telah diunduh pada langkah 1.
5. Tunggu proses berjalan hingga menampilkan hasil akurasi dan visualisasi.# UTS_NPL
